# House Price Prediction Using Machine Learning

## Project Description

This project predicts house prices using Machine Learning (Linear Regression) based on Area, Bedrooms, and Age of the house.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

- Scikit-Learn
- VS Code
